# Which tools to use

There are a variety of tools in the JMES Development Suite. This guide will help you find the right tools for your project.

If you are new to JMES, start with the [beginner's guide](./get-started.mdx).

## JMES.js <ThemedImage sources={{light:"/img/icons/jmes-js.svg", dark:"/img/icons/dark/jmes-js.svg"}} height="30px" />

Use JMES.js to create bots, mint NFTs, and build out back-end services. Visit the [JMES.js repository](https://github.com/jmesworld/jmes.js) to get started.


### Other Tools

The following tools allow you to interact with the JMES blockchain.

#### JMESd <ThemedImage sources={{light:"/img/icons/Terrad.svg", dark:"/img/icons/dark/Terrad.svg"}} height="20px" />

The command line interface and node daemon for interacting with the JMES blockchain. Use JMESd to run a full node or interact with the chain. Follow the [JMESd install guide](./jmesd/install-jmesd.mdx) to get started.

#### Explorer <ThemedImage sources={{light:"/img/icons/finder.svg", dark:"/img/icons/dark/finder.svg"}} height="20px" />

JMES’s multi-purpose [block explorer](http://explorer.jmes.art/).

#### Faucet <ThemedImage sources={{light:"/img/icons/Faucet.svg", dark:"/img/icons/dark/Faucet.svg"}} height="20px" />

Get tokens sent to your testnet address using the [JMES Testnet Faucet](http://faucet.jmes.art).
